
Настройка БД: config->db.php
Миграции:
для выполнения миграции в командной строке необходимо перейти в каталог cd migrations и выполнить php task_create.php
- Создание таблицы задач(php task_create.php)
- Добавление демо-данных в задачи(php task_demo_data.php)
- Создание таблицы пользователей(php user_create.php)
- Добавление демо-данных пользователей(php user_demo_data.php)
